# Isekai Guild Quests in Vanilla Tab

The Isekai Guild Faction Add-on posts its daily quests on a custom Quest Board reached through the faction's world-map settlement. This mod mirrors those daily listings straight into RimWorld's normal Quests tab as "Available" quests, so you can read and accept them without leaving your colony.

How it works: each in-game day the add-on rolls 5 board listings; this mod auto-posts them as offered vanilla quests (the same ones the board's "Accept" would create). Accepting a quest in the vanilla tab starts the hunt exactly as before. Because the listings are now auto-posted, the world-map board's own "Accept" button is disabled (it would create duplicates) and instead points you to the Quests tab; the board still works as a preview.
